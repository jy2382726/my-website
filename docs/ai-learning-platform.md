## 学习页 API 交互流

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│  1. 进入页面                                                   │
│     GET  /api/lessons/:id        → 课程内容 (Markdown)         │
│     GET  /api/progress/:id       → 学习状态                    │
│     GET  /api/chat/:lesson_id    → 历史对话                    │
│                                                                │
│  2. AI 对话 (SSE 流式)                                         │
│     POST /api/ai/chat                                         │
│     Body: { lesson_id, message }                              │
│     Response: SSE stream → 逐 token 渲染                      │
│                                                                │
│  3. 生成练习题                                                 │
│     POST /api/ai/exercise                                     │
│     Body: { lesson_id, difficulty }                            │
│     Response: { questions: [...] }                             │
│                                                                │
│  4. 标记完成                                                   │
│     PUT  /api/progress/:id     → { status: completed }        │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## AI 功能说明

### 对话辅导

- 每个课时可配置 `ai_prompt`（系统提示）
- AI 助手基于课程内容和系统提示进行辅导
- 对话历史持久化到 `chat_messages` 表
- 使用 SSE 流式响应，逐 token 渲染

### 练习题生成

- 管理员可为课时生成练习题
- 支持多种题型（选择、填空、代码）
- 难度可调（与课程难度联动）
- 练习题结果可记录到 progress.notes

### AI Prompt 设计

每个课时的 `ai_prompt` 字段控制 AI 对话行为：

```
示例 prompt:
"你是一个 AI 基础课程的辅导老师。当前课时讲的是 Transformer 架构。
请用通俗易懂的语言回答学生问题，适当使用类比。
如果学生问到超出范围的问题，引导他们回到本课主题。"
```

## 管理后台功能

### 课程管理 `/admin/courses`

- 课程列表（表格：标题/分类/难度/发布状态/操作）
- 新建/编辑课程表单
- 课时管理（关联到课程下）

### 课时管理 `/admin/courses/:id/lessons`

- 课时列表（支持拖拽排序）
- 新建/编辑课时：
  - 标题 + 排序序号
  - Markdown 编辑器（实时预览）
  - 课时类型选择（text/code/exercise）
  - AI Prompt 编辑（指导 AI 对话风格）
  - 发布状态控制

### 用户管理 `/admin/users`

- 用户列表
- 角色管理（设置 admin）

### 数据统计 `/admin/stats`

- 注册用户数
- 课程注册数
- 学习完成率
